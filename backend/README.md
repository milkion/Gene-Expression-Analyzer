# FIT3162 Bioinformatic Analysis GraphQL Backend

A GraphQL server implementation for handling genomic data analysis, built with Apollo Server and TypeScript.

## 📋 Prerequisites

- Node.js (v20.9.0 or higher)
- npm

## 🛠️ Installation

1. Clone the repository
2. Install dependencies by running `npm install`

<!-- Compile the code by running `npm run compile` -->
3. Run the server by running `npm start`
